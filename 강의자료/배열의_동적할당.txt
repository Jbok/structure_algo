c언어에서는 변수나 입력값이 변수 길이에 들어오지 못한다.
ex int n=10, int arr[n] <= 사용불가!

따라서 이러한 경우에는 배열을 미리 크게 만들어 놓는 방법도 있지만 이는 비효율적이기 때문에 동적할당을 통해서 배열의 크기를 동적으로 할당해서 사용하는 방법이있다.

1차원 배열
int *arr;
int arrsize;
arr = (int *)malloc(sizeof(int)*arrsize); //arr은 int의 포인터 형식이기 때문에 (int *) 동적할당으로 만들어 줘야하는 메모리의 크기는
int형과 그안에 들어갈 자료의 크기 이기 때문에 sizeof(int)*arrsize. 형식을 사용함.
